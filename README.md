# cubus.css
cubus.css adds animated (or static) 3d cubes to your website!

## Example

```html
<div class="cube__scene cube__size-3rem">
  <div class="cube cube__animation-xy-360">
    <div class="cube__face cube__face-red"></div>
    <div class="cube__face cube__face-red"></div>
    <div class="cube__face cube__face-red"></div>
    <div class="cube__face cube__face-red"></div>
    <div class="cube__face cube__face-red"></div>
    <div class="cube__face cube__face-red"></div>
  </div>
</div>
```
